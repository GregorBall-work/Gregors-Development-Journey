# Gregors-Development-Journey
A living record of my growth as a developer with the projects I build, the skills I’m learning, and the experiments that help me understand data more deeply. This showcases my personal development‑journey website, where I document progress, reflect on challenges, and share the ideas shaping my path.

🌱 Purpose of This Project
This site exists as both a portfolio and a learning journal. It helps me:

Track my progress across web development, data, and creative coding.

Build a transparent record of what I’m learning and why.

Share projects in a way that highlights process, not just outcomes.

Experiment with new technologies in a real, evolving environment.
